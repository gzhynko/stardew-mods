# Version history for Pufferfish Chickens

## 2.0.0
Released July 23, 2026.
- Rewrote the mod as a pure Content Patcher content pack (incl. compat with SDV 1.6).

## 1.0.4
Released August 13, 2021.
- Updated to support Harmony 2.1. SMAPI 3.12 or newer is now required.

Released May 26, 2020.
- Pufferfish eggs can now be incubated.

- The mod is now open-source.

## 1.0.1
Released May 25, 2020.
- Pufferfish chicken is now a new coop animal.
- Pufferfish eggs are now new custom objects.
- Modified the description to match the new features.
- Removed the [CP] folder as there is no need to replace anything now.

## 1.0.0
Released May 24, 2020.
